# ❤️ Heart Disease Prediction System

A machine learning-based web application that predicts the risk of heart disease using clinical data. This project is developed using **Data Mining techniques** and deployed with a Flask web interface.

---

## 📌 Project Overview

Heart disease is one of the leading causes of death worldwide. This project aims to analyze patient health data and predict whether a person is at risk of heart disease or not.

The system takes user input (age, cholesterol, blood pressure, etc.) and uses a trained machine learning model to classify the risk as:

* 🔴 High Risk (Heart Disease)
* 🟢 Low Risk (No Heart Disease)

---

## 🚀 Features

* User-friendly web interface
* Real-time prediction
* Machine learning classification model
* Data preprocessing and feature handling
* Clear result visualization (High / Low risk)

---

## 🛠️ Technologies Used

* **Python**
* **Flask** (Web Framework)
* **Scikit-learn** (Machine Learning)
* **Pandas & NumPy** (Data Processing)
* **HTML, CSS, JavaScript** (Frontend)

---

## 📊 Data Mining Concepts

This project is based on the following Data Mining techniques:

* **Classification** – Predicting heart disease (Yes/No)
* **Data Preprocessing** – Cleaning and preparing data
* **Feature Selection** – Identifying important attributes
* **Pattern Recognition** – Finding relationships in medical data

---

## ⚙️ How It Works

1. User enters medical data through the web form
2. Data is processed and converted into model input
3. Machine learning model predicts the result
4. Output is displayed as High Risk or Low Risk

---

## 📈 Model Performance

* Accuracy: **85%**
* Precision: **82%**
* Recall: **88%**
* F1-Score: **85%**

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/Sdia1029/Heart-disease-prediction.git
```

2. Navigate to project folder:

```
cd Heart-disease-prediction
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
python app.py
```

5. Open in browser:

```
http://127.0.0.1:5000/
```
